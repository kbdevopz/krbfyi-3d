# Karlis Baisden — 3D Portfolio

Interactive portfolio built with React, Three.js, and Tailwind CSS. Features a 3D hero scene, animated hexagonal skill grid, project showcase, and contact form.

Based on [sunnypatell/react-threejs-portfolio](https://github.com/sunnypatell/react-threejs-portfolio) (open source).

## Quick Start

```bash
npm install
npm run dev
```

## Environment Variables

Copy `.env.example` to `.env` and fill in:

- `VITE_EMAILJS_SERVICE_ID` — EmailJS service ID
- `VITE_EMAILJS_TEMPLATE_ID` — EmailJS template ID
- `VITE_EMAIL_JS_ACCESS_TOKEN` — EmailJS access token
- `VITE_RECAPTCHA_SITE_KEY` — Google reCAPTCHA site key

## Build

```bash
npm run build
npm run preview
```

## Tech Stack

- React 18, React Router 6, Vite
- Three.js, @react-three/fiber, @react-three/drei
- Tailwind CSS, Framer Motion
- EmailJS, reCAPTCHA
